LOL_em Ipsum_
=============

Random latin text getting you down?  Put on your lollerskates and install this gem (LOLZ).


Installation
------------

    $ gem install lolem

Install the lolem gem from [rubygems](https://rubygems.org/gems/lolem)


Usage (CLI)
-----------

    $ lolem 40

Generates 40 words of Lolem Ipsum (default is 30)


Usage (Ruby)
---------

    lolz = Lolem.new
    puts lolz.words 50
